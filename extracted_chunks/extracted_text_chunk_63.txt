the GENIVI4project The stack currently at version 21021 dated May 2018 consists of three main modules described in the following libvsomeipso the main library which is in charge of providing all the basic functionalities according to the protocol specifications It encompasses creation and serialization of message headers establishment of connections management of the communication endpoints and packets transmission more over it provides the event loop abstraction which simplifies the development of applications based on the vsomeip stack libvsomeipso however does not provide functionalities for the serialization of data structures com posing the payload the task is left to the application
